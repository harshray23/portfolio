# **App Name**: PortfolioPro

## Core Features:

- Profile Display: Display personal details (name, education, skills, etc.)
- Project Showcase: Showcase completed projects with title, description, and link to the GitHub repository.
- Contact Section: Add a contact form or display contact information (email, LinkedIn, etc.).
- Resume Download: Enable visitors to download your resume.
- AI Project Summarizer: Summarize project descriptions using generative AI; a tool will be used to intelligently truncate verbose content.
- Testimonial Section: Include testimonials or recommendations.
- About Me Section: Provide an 'About Me' section with a brief introduction.

## Style Guidelines:

- Primary color: Deep purple (#6A4CAF) to convey sophistication and technological expertise.
- Background color: Very dark gray (#222222) as a dark theme creates a modern, high-tech aesthetic.
- Accent color: Lavender (#D0BFFF) for highlights and interactive elements; contrasts against the purple, but remains harmonious.
- Font: 'Space Grotesk' sans-serif for titles and short amounts of body text; paired with 'Inter' for longer body sections.
- Use a set of line icons to represent different sections and skills, and provide a clean, professional look.
- Use a clean, responsive layout that adapts to different screen sizes. Incorporate white space to ensure readability.
- Add subtle animations for transitions and hover effects to enhance user experience.
